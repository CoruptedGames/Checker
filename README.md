# ✓ CHECKER

A modern **checklist / task tracker web app** built using **pure HTML, CSS, and JavaScript** with a premium **dark glassmorphism UI**.

CHECKER allows users to:

* Create custom checklists
* Import existing checklist files (`.json`)
* Track progress with checkboxes
* Save progress anytime
* Export updated checklists
* Continue later without losing progress

Designed to be simple, lightweight, and fully usable directly in the browser.

---

## ✨ Why This Project Exists

The idea behind **CHECKER** was simple:

Sometimes people want a checklist that is:

✅ Easy to create
✅ Reusable later
✅ Exportable as a file
✅ Editable anytime
✅ Works without an account or database

Examples:

* Grocery shopping 🛒
* Daily tasks ✅
* Gaming quests 🎮
* Study plans 📚
* Travel packing lists ✈️
* Minecraft server tasks ⛏️

Instead of using complicated apps, CHECKER keeps everything simple and stored locally in your browser or exported as a JSON file.

## 🚀 Features

### Create Checklists

Create your own checklist from scratch.

Add:

```txt
Milk
Bread
Cold Drink
Rice
```

and turn them into a reusable checklist.

---

### Import Existing Checklists

Already have a saved checklist?

Import a `.json` file and continue where you left off.

---

### Save Progress

Checkbox progress is saved automatically in the browser.

You can also export an updated JSON file anytime.

Example exported format:

```json
[
  {
    "name": "Milk",
    "checked": true
  },
  {
    "name": "Bread",
    "checked": false
  }
]
```

## 📖 How To Use

### 1. Create a Checklist

1. Open **CHECKER**
2. Click **Create New**
3. Add items
4. Click **Confirm & Export JSON**

A checklist file will be downloaded.

---

### 2. Import a Checklist

1. Open **CHECKER**
2. Click **Import Checklist**
3. Select your `.json` file
4. Continue checking items

---

### 3. Save Progress

Click:

```txt
Save Progress
```

to download an updated version of your checklist.
